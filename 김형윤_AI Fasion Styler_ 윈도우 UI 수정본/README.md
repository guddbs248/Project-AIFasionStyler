#프로젝트 제목: AI기반 옷 자동 추천 앱 
#개요: 저는 미리 준비하기보다는 일이 닥쳤을 때 시작하는 성격이라, 갑작스러운 상황에서 대응이 불안해지는 경험을 자주 했습니다. 특히 대부분의 사람들은 평일 이른 아침에 일어나 학교나 직장에 가야 하는 상황이 많은데, 저처럼 시간을 잘못 계산하면 버스를 놓치는 등 예상치 못한 불편을 겪기 쉽습니다. 이러한 경험에서 착안해 만든 것이 바로 AI 자동 옷 추천 웹 애플리케이션입니다.

이 애플리케이션은 사용자의 주간 일정, 날씨, 개인 옷장 데이터를 기반으로 맞춤형 코디를 제안합니다. 단순히 키워드를 매칭하는 수준을 넘어, AI 기반 가중치 추천 로직과 이미지 색상 자동 추출 기능을 활용하여 상황에 가장 적합한 스타일을 추천합니다.

매일 아침 무엇을 입을지 고민하며 시간을 낭비하는 사람들에게, 이 애플리케이션은 일주일치 옷차림을 미리 추천해주거나, 새로운 스타일 인사이트를 제공함으로써 고민 없이 바로 옷을 입을 수 있는 편리함을 제공하고 그들의 하루 3분 5분 시간을 절약해주는 것이 이 어플리케이션의 목표 입니다. 

#주요 기능:
1. 옷장관리
옷 등록 - 이미지 업로드가 가능하고, 옷 사진을 업로드하면 localStorage 용량을 절약하기 위해 Canvas를 사용, 고압축 jpeg로 자동 변환하여 저장합니다. 
AI 색상 추출: AI가 추출한 색상 중, 사용자가 배경색 등을 제외하고 실제 옷에 해당하는 색상만 직접 클릭하여 선택할 수 있습니다.

2. 옷장 조회 및 관리

세탁 상태 관리: 각 옷의 상태를 Ready -> Washing -> clean 으로 관리할 수 있습니다.
빨래 중인 옷은 코디 추천에서 제외 됩니다.

3. 일정 및 날씨 관리
월요일부터 일요일까지의 주간 캘린더에 일정을 텍스트로 입력할 수 있습니다.
날씨 및 상황에 대해 상세 문맥을 입력하고, 이 정보들은 코디 추천 시 가중치로 활용 됩니다.

4. Rule-based
가중치 기반 추천 엔진을 통해 script.js에 정의된 가중치 시스템을 기반으로 각 옷의 적합도 점수를 계산합니다.
추운 날씨(9도 이하)에서는 '아우터' 카테고리에 매우 높은 가산점을 부여하여 반드시 포함시킵니다.
공식 일정에는 정장, 셔츠 등에 가산점을 주고 캐주얼 의류에는 감점을 줍니다.
Canvas로 추출된 옷의 RGB 데이터를 HSV로 변환하여 상의-하의-아우터 간의 색상 조화를 계산해 점수에 반영합니다.
'이번 주에 입지 않은 옷'에 보너스 점수를 부여하여 다양한 옷을 추천하도록 유도합니다.
'원피스+하의'와 같이 부적합한 조합은 함수를 통해 필터링 됩니다.

#기술 스택: HTML, CSS, JavaScript, localStorage, Canvas API

#실행 방법: 
1. 파일의 index.html 열기
2. +옷 등록 버튼을 통해 본인이 가지고 있는 옷 등록
3. 메인에서 추가된 옷들을 확인
4. 일주일치 일정을 작성하고 '코디 추천 받기' 실행
5. 옷 추천 결과 확인

#기타 참고사항:
저장 공간 제한: localStorage는 브라우저별로 5MB~10MB의 용량 제한이 있습니다. 본 과제물은 이미지 압축 기능과 용량 초과 시 오래된 이미지부터 자동 삭제하는 기능으로 이를 대비하고 있습니다.
